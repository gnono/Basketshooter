Ziel: Die gelbe Münze im 3. Stock abzuholen und auf eine Platform im 1. Stock zu legen.
Sobald das Spiel startet, wird ein Timer ebenfalls gestartet.
Der Spieler hat die Chance seinen eigenen Rekord zu brechen, indem er das Spiel schneller als vorhin abschließt.  

Gameplay: 
    -Objekte zerstören: linke Maustaste
    -Objekte aufheben: rechte Maustaste
    -Charakterbewegung: Mit WASD Tasten
    -Slowmotion-Effekt: T Taste
    -Treppe platzieren: R Taste
    
Arsenal:
    -Kugeln
    -Bomben
    
Zerstörung von Objekten: 
    -Kugeln zerstören nur Würfeln.
    -Bomben zerstören Objekte, die man selbst erstellt hat (Treppen).
    
Bemerkungen:
    -Alle Objekte werden in die Luft gesprengt, sobald sie im Explosionsradius einer Bombe sind.
    -Treppen werden erst zerstört, wenn eine Bombe damit kollidiert.
